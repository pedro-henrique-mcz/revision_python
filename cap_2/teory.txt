2. Explique a diferença entre os três métodos de string a seguir: .title(), .upper() e .lower().

.title() = Coloca em caixa alta cada primeira letra de uma palavra de uma string. .upper() coloca toda a string em caixa alta. .lower() coloca toda a string em caixa baixa.

4. Imagine que você quer exibir uma mensagem que inclui a idade de uma pessoa, armazenada em uma variável idade = 29. Por que o código print("A idade é: " + idade) causaria um TypeError? Como você corrigiria esse código usando a função str()?

Type erro sobe por diversas maneiras, mas a mais comum [e quando tentamos fazer uma operação ou expressão com tipos incompatíveis, no caso do exemplo, quando usamos o sinal de soma em uma expressão que tem uma string envolvida o que acontece é que o python interpreta aquilo como uma concatenação de strings e espera que todos os envolvidos na expressão sejam string, como idade é um int, o erro sobe. A correção se dá por uma convesão de tipo com a função str(idade).
