1. Qual é a finalidade da extensão de arquivo .py? O que ela indica para o seu editor de texto e para o sistema?

A extensão ou sufixo py indica que o arquivo contem um modulo python e que deve ser interpretado pelo interpretador python.

5. Quando um programa Python não executa corretamente por causa de um erro, o que é um traceback e por que ele é útil para resolver o problema?

Traceback são a maneira que o interpretador python tem de te devolver erros para serem analisados, trace backs devolvem o arquivo do erro, a linha do erro e o tipo do erro, pode acontecer uma cascata caso haja partes dependentes no codigo.
